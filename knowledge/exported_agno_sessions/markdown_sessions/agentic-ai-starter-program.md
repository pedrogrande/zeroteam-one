# Agentic AI Starter Program — Course Outline & Learning Outcomes

***

## Course Overview

A 6-week, hands-on program where learners build a working agentic application every week using Agno AgentOS and VS Code Copilot. No prior coding experience required.

**Delivery:** Weekly 90-min live webinars, hosted dev environment, Telegram cohort group, weekly homework challenges, final video presentation.

***

## Module Outline

### Module 1: Your First Agno Agent

*Build and interact with your first AI agent from scratch.*

- Define what an AI agent is, including its core features and components
- Identify the strengths and limitations of agentic systems
- Navigate the Agno AgentOS development environment confidently
- Build and run a basic Agno agent using VS Code Copilot assistance
- Add a tool to an agent to enable it to take action

***

### Module 2: Smarter Agents

*Give your agent access to knowledge and context.*

- Explain the role of vector databases, memory, and sessions in agent intelligence
- Set up a knowledge base and connect it to an agent using Agno tools
- Implement a basic RAG (Retrieval-Augmented Generation) pipeline
- Apply "minimum sufficient context" principles to keep agents focused and efficient
- Configure context providers and search tools for richer agent responses

***

### Module 3: Multi-Agent Systems

*Coordinate teams of agents working together.*

- Describe common multi-agent use cases and team modalities
- Design agent teams with clear roles and responsibilities
- Implement and compare different Agno team modes (routing, broadcast, tasks)
- Evaluate multi-agent team outputs and identify design improvements
- Integrate a human interface (Telegram or Slack) with an agent team

***

### Module 4: Agentic Workflows

*Build reliable, structured autonomous workflows.*

- Distinguish between agentic workflows and agent/team architectures, and know when to use each
- Design workflows with well-defined structured inputs and outputs
- Implement a working agentic workflow using Agno workflow primitives
- Explain the role and value of Human-in-the-Loop (HITL) design patterns
- Build a HITL workflow with at least one human approval or review gate

***

### Module 5: Agentic Workflow Design

*Design workflows with professional-grade principles.*

- Articulate the CAWDP framework and apply its principles to workflow design
- Audit an existing workflow and identify gaps against CAWDP criteria
- Redesign and implement an improved workflow grounded in CAWDP principles
- Test and evaluate agentic workflows systematically using logging and tracing
- Document design decisions clearly to support review and iteration

***

### Module 6: Final Project

*Build a complete agentic system for a real problem.*

- Scope and plan an original agentic system from problem definition to implementation
- Integrate agents, knowledge, tools, and workflows into a coherent system design
- Apply the skills and principles developed across all five previous modules
- Test the system and evaluate its performance against intended outcomes
- Present the project confidently as a recorded video walkthrough covering design, build, and results

***
