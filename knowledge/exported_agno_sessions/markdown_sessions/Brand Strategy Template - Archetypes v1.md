# Brand Strategy Template \- Brand Archetype

This template is designed to guide entrepreneurs through the process of developing a strong and compelling brand archetype. 

By answering the following questions, you will gain clarity on your brand's purpose, identity, and positioning, which will help you build a successful and sustainable business.

Once completed, you will be able to determine your brand archetype.

| Key Questions | Your Answers |
| :---- | :---- |
| **1\. Mission and Vision** |  |
| What is the core mission of your startup? |  |
| What kind of impact do you want your brand to have on your customers and the world? |  |
| What vision drives your team every day? |  |
| **2\. Target Audience** |  |
| Who is your ideal customer, and what are their key needs, desires, and values? |  |
| How do you want your customers to feel when they interact with your brand? |  |
| What role do you want your brand to play in your customers' lives? |  |
| **3\. Brand Personality and Values** |  |
| If your brand were a person, how would you describe its personality (e.g., bold, nurturing, creative)? |  |
| What are the core values your startup stands for? |  |
| How would you like people to describe your brand in one word? |  |
| **4\. Positioning and Differentiation** |  |
| What makes your startup unique compared to competitors? |  |
| What do you want people to remember most about your brand? |  |
| Are there any brands you admire and want to emulate (or avoid)? |  |
| **5\. Customer Journey** |  |
| What emotional experience do you want customers to have throughout their journey with your product/service? |  |
| What kind of story do you want your brand to tell? |  |
| Do you see your brand as solving a problem, inspiring action, or connecting people? |  |
| **6\. Industry and Market** |  |
| How do you want your startup to be perceived within your industry? |  |
| Is your industry driven by innovation, tradition, luxury, or something else? |  |
| Are there common archetypes in your market you want to stand out from or align with? |  |
| **7\. Cultural and Emotional Appeal** |  |
| Are there specific emotions or aspirations you want your brand to evoke? |  |
| How do you want your brand to influence culture or be influenced by it? |  |
| What’s your brand’s stance on societal issues, if any? |  |
| **8\. Long-Term Goals** |  |
| What’s the long-term vision for the brand? |  |
| How do you see it evolving? |  |
| How do you want customers to describe your brand 10 years from now? |  |
| What legacy do you want your brand to leave? |  |
| **9\. Visual and Verbal Identity** |  |
| What kind of visuals (e.g., colors, styles) do you feel represent your brand? |  |
| How would your brand "speak" to its customers? (e.g., formal, playful, inspirational) |  |
| Do you want your tone to challenge, educate, entertain, or comfort? |  |
