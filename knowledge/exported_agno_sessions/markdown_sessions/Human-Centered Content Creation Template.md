# Human-Centered Content Creation Template

This template guides you through a human-centered design approach to content creation, ensuring your work resonates with your audience and delivers a meaningful message. Whether it's written, audio, or video content, this template helps you craft impactful and memorable experiences.

# Goals and Outcomes

| Define Goals and Desired Outcomes | Answers and Insights |
| :---- | :---- |
| Why am I creating this content? What is the purpose or goal? |  |
| What outcome do I hope to achieve with this content? |  |
| How will I measure the success of this content? |  |
| How can I encourage my audience to stay engaged and consume the entire piece of content? |  |

# Audience Identification and Empathy

| Questions to Understand Your Audience | Answers and Insights |
| :---- | :---- |
| Who is my target audience? |  |
| What are their demographics (age, gender, location, education, etc.)? |  |
| What are their interests, needs, and pain points? |  |
| What motivates them? |  |
| What challenges do they face? |  |
| How do they currently consume content? |  |
| What are their expectations from content like this? |  |
| What are their potential objections or concerns? |  |

# Key Message and Context

| Questions to Define Your Message and Context | Answers and Insights |
| :---- | :---- |
| What is the core message I want to convey? |  |
| What do I want my audience to **feel** by engaging with this content? |  |
| What do I want my audience to **think** by engaging with this content? |  |
| What do I want my audience to **do** after engaging with this content? |  |
| What information or research supports my message? |  |
| What context is necessary for my audience to understand the message? |  |
| How does my message relate to my audience's interests and needs? |  |
| How can I frame my message to resonate with my audience's values and beliefs? |  |

# Content Structure and Narrative

| Main Points and Narrative |  |
| :---- | :---- |
| What are the main points you want to cover in your content? |  |
| What is a narrative that weaves these points together in a compelling way? |  |
| Can the narrative build tension, create empathy, or evoke emotion? |  |
| Does the narrative align with your key message and resonate with your audience? |  |

# Adding the "Special" Touch

| Questions to Enhance Your Content | Answers and Insights |
| :---- | :---- |
| How can I make this content unique, memorable and **fun**? |  |
| How can I surprise or delight my audience? |  |
| How can I provide value beyond the core message? |  |
| How can I incorporate storytelling, humor, or personal experiences? |  |
| How can I make my audience feel heard and understood? |  |
| How can I create a sense of community or connection? |  |

# Conclusion

By following this human-centered content creation template, you can develop content that not only informs but also inspires, entertains, and fosters a genuine connection with your audience. Remember, the goal is to create content that your audience will be happy to receive and find valuable.
