# Founder's SWOT Analysis Template

This template is designed to assist founders in conducting a comprehensive SWOT analysis for their organization. By identifying internal strengths and weaknesses, as well as external opportunities and threats, founders can gain valuable insights to inform strategic decision-making and drive business growth.

# Strengths (Internal)

| Question | Answer |
| :---- | :---- |
| What are the top personal strengths of the founder/s and team? (Complete a [strength profile](https://www.strengthsprofile.com/) to find out) |  |
| What does your organization do exceptionally well? (Focus on core competencies or unique offerings.) |  |
| What resources, skills, or expertise give your organization an edge? (Consider talent, technology, or intellectual property.) |  |
| What advantages does your product or service have over competitors’? |  |
| What do customers consistently praise about your organization? (Customer feedback, testimonials, or recurring positive themes.) |  |
| What partnerships or relationships enhance your organization’s success? |  |

# Weaknesses (Internal)

| Question | Answer |
| :---- | :---- |
| What internal challenges or limitations does your organization face? (E.g., skill gaps, outdated systems, or lack of resources.) |  |
| What feedback or complaints do you receive from customers? (Identify areas needing improvement.) |  |
| Are there processes or operations that are inefficient or costly? |  |
| What prevents your organization from scaling or growing faster? (Barriers like funding, talent, or infrastructure.) |  |
| How does your organization compare to competitors in terms of weaknesses? (What are they doing better?) |  |

# Opportunities (External)

| Question | Answer |
| :---- | :---- |
| What trends from the PESTEL analysis could create opportunities for growth? (E.g., emerging technologies, market demands, or cultural shifts.) |  |
| What unmet needs or underserved markets can your organization address? |  |
| Are there partnerships or collaborations you can leverage to expand your reach? |  |
| How can your organization take advantage of competitor weaknesses? |  |
| What changes in customer behavior or preferences present new opportunities? |  |

# Threats (External)

| Question | Answer |
| :---- | :---- |
| What trends from the PESTEL analysis could pose risks to your organization? (E.g., regulatory changes, economic downturns, or technological disruptions.) |  |
| Who are your main competitors, and what threats do they pose? |  |
| Are there potential market shifts that could reduce demand for your offerings? |  |
| What external dependencies (e.g., suppliers, partners) could disrupt operations? |  |
| What risks, such as cybersecurity threats or reputational damage, could harm your organization? |  |

## SWOT Analysis Worksheet

| Question | Findings |
| :---- | :---- |
| **Strengths (Internal, Positive Factors)** |  |
| What unique advantages does your business have? |  |
| What do you do better than anyone else? |  |
| What unique or lowest-cost resources can you draw upon that others can't? |  |
| What do people in your market see as your strengths? |  |
| What factors mean that you "get the sale"? |  |
| What is your company’s Unique Selling Proposition (USP)? |  |
| What proprietary technology or processes do you have? |  |
| What is strong about your team's expertise and experience? |  |
| What positive feedback have you received from customers or partners? |  |
| What valuable assets do you possess (e.g., intellectual property, strong brand, established customer base)? |  |
|  |  |
| **Weaknesses (Internal, Negative Factors)** |  |
| What could you improve? |  |
| What should you avoid? |  |
| What are people in your market likely to see as weaknesses? |  |
| What factors lose you sales? |  |
| What resources do you lack (e.g., funding, expertise, technology)? |  |
| What areas of your business are underperforming? |  |
| Are there gaps in your team's skills or experience? |  |
| What negative feedback have you received? |  |
| Are there any internal processes that are inefficient or ineffective? |  |
| What is your business lacking (e.g., a clear strategic direction, a strong brand)? |  |
|  |  |
| **Opportunities (External, Positive Factors)** |  |
| What good opportunities can you spot? |  |
| What interesting trends are you aware of? |  |
| Are there any underserved markets or customer needs? |  |
| Are there upcoming changes in technology that you can leverage? |  |
| Are there changes in government policy or regulations that could benefit your business? |  |
| Are there any social or lifestyle changes creating new possibilities? |  |
| Could any of your strengths open up new opportunities? |  |
| Are there any global influences or new markets you could explore? |  |
| Are there potential partnerships or collaborations that could be beneficial? |  |
| Can you identify any competitor weaknesses that you can exploit? |  |
|  |  |
| **Threats (External, Negative Factors)** |  |
| What obstacles do you face? |  |
| What are your competitors doing? |  |
| Are quality standards or specifications for your job, products or services changing? |  |
| Is changing technology threatening your position? |  |
| Do you have bad debt or cash-flow problems? |  |
| Could any of your weaknesses seriously threaten your business? |  |
| What economic trends (e.g., recession, inflation) could impact your business? |  |
| Are there new competitors entering your market? |  |
| Are there changes in consumer behavior or preferences that could negatively affect you? |  |
| Are there any political or legal changes that could create challenges? |  |
