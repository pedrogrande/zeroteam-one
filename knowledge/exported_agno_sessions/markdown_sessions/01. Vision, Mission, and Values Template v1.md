# Vision, Mission, and Values Template

This template is designed to guide organizations in defining their vision, mission, and values. 

Each section includes key questions to stimulate thought and discussion, helping to articulate a clear and compelling direction for the future.

Be sure to be thinking of the best case scenario \- imagine everything goes to plan \- even beyond your wildest dreams\!

# Vision

| Question | Response |
| :---- | :---- |
| What is the organization doing in 5 years? (products, services, markets) |  |
| What does the organization look like in 5 years? (size, structure, culture) |  |
| What does the world look like in 5 years when your organization is achieving its mission? (impact, influence) |  |
| How has the organization changed the lives of its customers (or stakeholders)? |  |
| What is the simplest way to describe the organization’s vision for the future? |  |

# Mission

| Question | Response |
| :---- | :---- |
| What problem does the organization exist to solve? |  |
| Who does the organization serve? |  |
| What is the organization’s main activity or focus? |  |
| What measurable outcomes does the organization aim to achieve? |  |

# Values
