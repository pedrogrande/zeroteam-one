# Agentic AI Starter Program

I am developing an agentic AI short course I can offer to the public. I realised I really enjoy helping people start their journey into the space and opening their eyes to what's possible.

I am using Agno as the core tool along with VS Code Copilot tools so that learners don't need a tech background or knowledge to take part.

It's designed to be a very practical program where every week we create a new agentic 'app'.

## Course delivery

1. Weekly, 90 minute live, code-along webinars
2. A hosted agent development environment (or set up your own local environment).
3. Weekly reflection exercises and challenges to share with the cohort. Guides and prompts provided for learning between webinars.
4. Telegram group for cohort communications, peer-to-peer support, knowledge sharing
5. Present your final project as a video presentation explaining what it does, how it was created, results of testing, etc

## Modules

1. Your first Agno agent - learning agent with Agno AgentOS. Homework: create a personal assistant agent
2. Smarter agents - giving the agent a range of knowledge bases using different tools available with Agno. Homework: set up a RAG system for your agent
3. Multi-agent systems - basic agent team trying some different teamwork modes. Homework challenge: Build a routing, broadcast or tasks mode agent team.
4. Agentic workflows - build a simple autonomous workflow with structured inputs and outputs. Homework challenge: Human-in-the-loop workflows - build a HITL workflow
5. Agentic workflow design - redesign a workflow using CAWDP principles and framework. Homework challenge: build an agentic workflow with CAWDP principles.
6. Final project: build your own agentic system for fun and profit.

Additional options

1. Human Interfaces for agents, teams and workflows (Slack, Telegram, Web AG-UI, A2A)
2. Deploy to Railway
3. Build a portfolio site for your agent projects
